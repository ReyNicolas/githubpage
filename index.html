The mushroom legacy (2025-2026) (C# Unity) (Concluded Collaboration)

Itch.io link: https://aysunza.itch.io/the-mushroom-legacy

Collaborative Unity project where I was the sole Unity Dev from the beginning until March 2026.
The project started from a basic controller (MoreMountains, only running) and was scaled into a complete ecosystem of interconnected systems: stamina-based player abilities, inventory, liquids, dungeon generation, and enemy AI.
I decided to conclude my collaboration and step away from the project.

Project Contributions (Sole Unity Gamedev):

  • Procedural Dungeon Generator: Created a system that generates maps using fully procedural rooms or predefined templates connected by valid doors.
  • Inventory & Equipment System: Developed an inventory with mountable storage and item actions (drop, use, equip, unequip, transfer), fully integrated with equipment and item usage logic.
  • Dual Grid Liquid System: Built an expansion-ready liquid simulation featuring directional flow, wall/abyss interactions, visual effects when falling, and entity/object interaction.
  • Liquid Detection: Implemented a system that detects liquids under the player and entities, replacing normal movement effects (run, jump, dash) with custom particle feedback.
  • Player Controller & Abilities: Programmed movement mechanics (running, jumping, dashing, pushing) governed by stamina costs and a capability-blocking logic flow.
  • Object Handling & Aiming: Developed logic for grabbing, releasing, and carrying objects on the head, including a forced-throw trigger when hit or dashing, and directional aiming for throwing items with custom hit logic per item.
  • Corpse Controller: Designed a system that manages dynamic bodies using modular components to execute specific actions upon generation (such as custom storage components, visual display updates, or auto-destruction timers).
  • Environment Elements & Props: Programmed interactive hazards and items, including chests (unlocking, setting contents, and inventory interaction), spike traps, pushable items, collectible pushables, and barrels that release damage or liquids.
  • Safe Zone Finder: Built a positioning tool to find valid, safe areas for object and prop placement.
  • Abyss System: Engineered an optimized checking system for entities, objects, and liquids falling into the abyss.
  • Enemy AI & Slime Logic: Implemented enemy pathfinding for patrolling, chasing, and attacking states, alongside a special slime behavior that handles division on death, item absorption, and movement logic to actively avoid non-slime liquids.
  • Status Effects: Developed a status effects logic applicable to both the player and external entities.
  • Game Feel & Juicing: Polished game feel, visual feedback, by scripting smooth tweens and feedback triggers using DOTween and Feel (More Mountains).
